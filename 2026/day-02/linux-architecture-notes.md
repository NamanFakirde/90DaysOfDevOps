<h3>Processes</h3>
<p>
  <b>Everything starts with a process in linux."</b><br>
  Below are the states of the process in linux:
  <ol>
    <li>Running: Actively using the CPU or waiting for a turn.</li>
    <li>Sleeping: Waiting for an event; can be interrupted.</li>
    <li>Killed: Paused manually by the user or by a debugger.</li>
    <li>Zombie: Occurs when a process has finished its execution but its entry still remains in the Process Table.</li>
  </ol>
</p>

<h3>Linux commands</h3>
basic linux commands which is used frequently- 
<ol>
  <li>ls: list everything in the current directory.</li>
  <li>cd: change the directory. </li>
  <li>rm: Use to remove/delete a file (rm -r folder_name: to delete the directory).</li>
  <li>cp: To copy file from source to destination. </li>
  <li>mv: This command is used to rename a file and move a file/dirctory.</li>
</ol>
